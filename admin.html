Variantes del Producto
Producto Principal como Tono:

El producto principal (nombre, imagen, stock) se convierte automaticamente en el primer tono disponible. Puedes editar su nombre de tono y color abajo.

Consejos para variantes
  • El producto principal siempre es el primer tono - usa el nombre del producto como nombre de tono o personalizalo
  • Cada variante adicional debe tener un nombre unico (ej: "Rubio Dorado", "Rojo Carmesi")
  • Selecciona el color representativo con el selector de color
  • El cliente vera una manchita de este color para elegir el tono
  • Gestiona el stock de cada variante por separado

Imagen principal del producto. Si tiene variantes, esta sera la imagen del tono principal.

Consejos Utiles
  • Los productos se guardan automaticamente en el navegador
  • Usa imagenes cuadradas para mejor resultado
  • Puedes subir fotos directamente desde tu celular
  • Los cambios se ven inmediatamente en la tienda
  • Usa fotos con buena iluminacion para que los productos se vean mas atractivos
  • Escribe descripciones cortas explicando para que sirve cada cosa
  • Actualiza los precios cada vez que cambien
  • Revisa la cantidad de productos para evitar vender algo que no tengas
  • NUEVO: El producto principal ahora cuenta como un tono/variante
